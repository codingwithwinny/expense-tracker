# Expense Tracker

## Overview
This project is a comprehensive expense tracker that allows users to manage their budget effectively and visually represent their spending through interactive charts.

## Features
- **Dark/Light Theme Toggle:** Users can switch between dark and light themes for a better user experience based on their preferences.
- **Full SaaS Dashboard Redesign:** The dashboard has been redesigned to enhance usability with a modern interface.
- **Pie Charts:** Visual representation of spending categories using interactive pie charts.
- **Budget Progress Bars:** Users can set budgets and track their progress visually through progress bars.
- **Enhanced Dialog System:** The dialog system has been improved for better user interaction and feedback.
